### event loop (事件循环/事件轮询)
  - JS是单线程运行的
  - 异步要基于回调函数来实现
  - event loop 就是异步回调的实现原理

### JS是如何执行代码的？
```
从前到后，一行一行的执行，如果某一行代码执行错误，则停止下面代码的执行，并且是先把同步代码执行完，再执行异步代码
```
示例
```
console.log('Hi')

setTimeout(function cb1() {
 console.log('cb1') // cb 即callback
}, 5000)

console.log('Bye')

```
