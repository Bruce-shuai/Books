# TypeScript vs JavaScript
TypeScript是“强类型”版的JS(JS是弱类型语言)，当我们在代码中定义变量(包括普通变量、函数、组件、hook等)的时候，TS允许我们在定义的同时指定其类型，这样使用者在使用不当的时候就会被及时报错提醒
